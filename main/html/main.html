¿Quienes Somos?

Somos Computer Innovations, el proveedor líder de soluciones de software open source diseñadas especificamente para escuelas e institutos.

Nuestra misión es brindar tecnología accesible, eficiente y segura para optimizar la gestión y el aprendizaje.

¿Que ofrecemos?

Ofrecemos un completo ecosistema de herramientas dijitales:

- Sistema Operativo Educativo.

- Servicios de Software.

- Portal de Descargas.

- Sistema de Ticketing.
